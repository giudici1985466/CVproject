# CVproject

Repository for the Computer Vision a.y. 2025/26 project.

## Robust AI-Generated Image Detection

This repository contains the code and materials for a computer vision project on detecting **AI-generated images** under realistic post-processing conditions.

The project studies **real/fake classification** across three image versions:

* original
* transfer
* redigital

It includes experiments with:

* ResNet18
* single-patch and multi-patch evaluation
* joint multi-task learning
* patch disagreement analysis

The original dataset is available at:
https://zenodo.org/records/14963880

The procedure used to obtain the reduced subset is included in the project notebook.

The final presentation included in the repository reports the main results and conclusions.
